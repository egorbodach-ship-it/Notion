# Сборка .jar в облаке через GitHub Actions

Этот способ **не требует установки Java и Maven** на твой компьютер — всё собирается на серверах GitHub.

## Шаги

1. Зайди на https://github.com и создай новый репозиторий (кнопка **New**).
   - Имя любое, например `SimpleEssentials`. Можно приватный (Private).
   - НИЧЕГО не добавляй при создании (ни README, ни .gitignore) — просто Create repository.

2. Загрузи файлы проекта в репозиторий:
   - На странице репозитория нажми **Add file -> Upload files**.
   - Перетащи туда СОДЕРЖИМОЕ папки SimpleEssentials (файлы pom.xml, папки src и .github).
   - ВАЖНО: папка `.github/workflows/build.yml` обязательно должна сохраниться (её легко потерять при drag&drop — проверь, что она есть).
   - Нажми **Commit changes**.

3. Сборка запустится автоматически. Открой вкладку **Actions** в репозитории.
   - Дождись, пока у задачи «Build plugin» появится зелёная галочка (около 1–2 минут).

4. Скачай готовый .jar:
   - Открой завершённый запуск в Actions.
   - Внизу в разделе **Artifacts** будет файл `SimpleEssentials-jar` — скачай и распакуй.
   - Внутри — `SimpleEssentials-1.0.0.jar`. Это и есть готовый плагин.

5. Кинь .jar в папку `plugins` своего сервера Minecraft 1.20.1 и перезапусти его.

Готово!
